# Binance Futures Trading Bot 🚀

## 📌 Overview

This project is a Python-based trading bot that interacts with Binance Futures Testnet.  
It allows users to place MARKET and LIMIT orders via CLI and UI.

---

## ⚙️ Features

- Place MARKET and LIMIT orders
- Supports BUY and SELL
- CLI interface (argparse)
- Streamlit UI (bonus feature)
- Logging system
- Error handling

---

## 🛠️ Tech Stack

- Python
- python-binance
- Streamlit
